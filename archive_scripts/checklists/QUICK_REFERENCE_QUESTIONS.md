# Weekly Client Check-In - Quick Reference Card

**Print this and keep it handy during client calls**

---

## THE 10 ESSENTIAL QUESTIONS

### 1. PROGRESS
- [ ] Are we on track with this week's deliverables?
- [ ] Any delays or roadblocks affecting progress?

### 2. SCOPE (Critical - Don't Miss!)
- [ ] Has anything new come up since last week?
- [ ] Are we doing anything that wasn't in the original scope?
- [ ] Have you been thinking about any additional work or studies?

### 3. TECHNICAL BLOCKERS
- [ ] Are we waiting on any information or data from you?
- [ ] Any issues with the network operator or third parties?
- [ ] Any technical challenges we should discuss?

### 4. CLIENT SATISFACTION (Be Observant)
- [ ] How are you feeling about progress?
- [ ] Is our communication frequency working for you?
- [ ] Any concerns or frustrations we should address?
- [ ] Anything you're particularly happy with?

### 5. SCHEDULE
- [ ] Are the current timelines still working for you?
- [ ] Any external deadlines or pressures we should know about?
- [ ] Need anything accelerated or can anything wait?

### 6. BUDGET/COMMERCIAL
- [ ] Project tracking to your expectations cost-wise?
- [ ] Any questions about scope vs. budget?

### 7. UPCOMING ISSUES
- [ ] Anything coming up in the next 2 weeks we should prepare for?
- [ ] Any decisions you'll need to make soon?
- [ ] Any risks you're worried about?

### 8. ACTIONS
- [ ] What do you need from us this week?
- [ ] What information or decisions do we need from you?
- [ ] Anyone else we need to coordinate with?

### 9. COMMUNICATION
- [ ] Is our current meeting frequency working?
- [ ] Prefer email, calls, or face-to-face for updates?

### 10. OVERALL HEALTH CHECK
- [ ] On a scale of 1-10, how do you feel about the project?
- [ ] Anything else on your mind we haven't covered?
